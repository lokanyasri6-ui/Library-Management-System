# 📚 Library Management System

A simple Java-based Library Management System developed using Object-Oriented Programming (OOP) concepts. This project allows users to manage books and members, issue books, return books, and view library records through a console-based application.

## 🚀 Features

- Add books to the library
- Add members to the library
- Display all books
- Display all members
- Issue books to members
- Return issued books
- Track book availability
- Exception handling for invalid operations

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- IntelliJ IDEA

## ⚙️ Installation & Setup

### Prerequisites

Make sure the following are installed:

- Java JDK 8 or higher
- IntelliJ IDEA (or any Java IDE)
- Git

### Clone the Repository

bash
git clone https://github.com/lokanyasri6-ui/Library-Management-System.git


### Navigate to the Project Folder

bash
cd Library-Management-System


### Open in IntelliJ IDEA

1. Open IntelliJ IDEA.
2. Click *Open*.
3. Select the project folder.
4. Wait for the project to load.

### Run the Application

1. Open LibraryManagement.java or Main.java.
2. Click the *Run ▶️* button.
3. View the output in the console.

### Compile and Run Using Terminal

bash
javac src/*.java
java -cp src Main


### Verify Java Installation

bash
java --version
javac --version
